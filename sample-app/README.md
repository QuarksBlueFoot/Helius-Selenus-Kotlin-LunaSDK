# Helius Luna SDK Sample App

This is a simple Android application demonstrating the usage of the LunaSDK.

## Setup

1.  Open this project in **Android Studio**.
2.  Sync Gradle.
3.  Run on an emulator or device.

## Features

-   **Fetch Assets (DAS)**: Retrieves assets for a demo wallet using the Digital Asset Standard API.
-   **Validator Info**: Fetches current network slot and block height using generic RPC calls.
-   **Sender Transaction**: Simulates sending a transaction via the Helius Sender service (demonstrates the API call, though the transaction is dummy).

## Configuration

The API key is hardcoded in `MainActivity.kt` for demonstration purposes.
